# 설정

## 💰 주요 기능
- 요금 설정: 시간 간격당 요금 설정
- 방 갯수 설정: 방 생성 갯수 조절
- 실시간 업데이트: 설정 변경 즉시 적용

## 🔧 기술
- 데이터 소스: `users_{countryCode}` 컬렉션
- 자동 계산: 시간당 요금 자동 계산
- 일괄 업데이트: 모든 방 카드 요금 동시 업데이트

## 🔄 자동화
- 요금 계산: (ratePerInterval * 60) / timeInterval
- 방 생성: 설정된 갯수만큼 자동 생성
- 요금 동기화: 모든 방 카드 요금 일괄 업데이트

## 🎨 UI
- 숫자 입력 필드
- 저장 상태 표시
- 실시간 미리보기
