# 시스템

## 🏠 주요 기능
- 방 관리: 방 생성, 상태 관리, 요금 설정
- 게임 운영: 게임 시작/종료, 실시간 시간 계산
- 결제 시스템: 자동 요금 계산 및 정산
- 실시간 모니터링: 방 상태 및 시간 업데이트

## 🔧 기술
- 데이터 소스: `users_{countryCode}` 컬렉션
- 실시간 리스닝: Firestore onSnapshot
- 요금 계산: 1초마다 실시간 업데이트

## ⏰ 요금 로직
- 시간 간격 요금 (예: 10분당 5000원)
- 완성된 시간 간격만 요금 부과
- 실시간 요금 표시

## 🎨 UI
- 카드형 방 관리
- 실시간 시간 표시
- 결제 모달
- 상태별 색상 구분
